1. Information We Collect

  • Account credentials: Admins log in using their email and password via Firebase Authentication.
  • Biometric login: If enabled, biometric data (fingerprint or face) is used for quick access. This is processed locally by your device and never stored by us.
  • FCM Token: A Firebase Cloud Messaging token is used to send push notifications about orders and system alerts.
  • Security PIN: A 4-digit PIN can be set to secure sensitive areas like KBZ management. This is stored securely in Firestore.
  • KBZ Account Info: Includes bank numbers, KPay QR images, and nicknames managed by admins. Saved in Firestore and Firebase Storage.
  • Price Manager: Values and calculations entered for pricing are stored with timestamps in Firestore.
  • Daily Balance: Admins can input and store daily balance data in Firestore.
  • Order Management: The app lets admins view and update user orders, statuses, and upload payment confirmation screenshots.

2. How We Use Your Information

  • To authenticate and allow admin access.
  • To manage user orders and update statuses.
  • To send order alerts and admin notifications.
  • To secure sensitive areas using biometric login and PIN codes.

3. Data Security

We use Firebase security features such as Authentication, Firestore Rules, and Storage Rules to protect your data.

4. Data Sharing

We do not share your personal data with third parties, except:

  • When required by law.
  • With trusted services like Firebase (used as backend infrastructure).

5. Policy Updates

Any changes to this policy will be reflected here. We may also notify you via the app for major updates.
